Machine Learning Project

In this project I trained a neural network to predict O-ring volume swell for sustainable aviation fuels.

See the full publication here.

Data preprocessing in R

Prepared the raw data for use in a neural network. This involved eliminating features with low variance, removing correlated features, removing instances with missing values, and scaling the data.

Neural network development with H2O Flow

A preset grid of neural networks were trained and scored, followed by the random generation of neural networks using the AutoML feature in H2O Flow. The result was a neural network consisting of two hidden layers with 500 neurons each. Rectifier activation was used for the hidden layers with a dropout ratio of 0.4 for regularization.

Optimization in Python

The general approach was to use ant colony optimization in Python to maximize fuel performance within operability limits. Only the final plotting code is available here do to confidentiality requirements.
